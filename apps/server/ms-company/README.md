## Swagger-UI
- To use swagger, build the project and access this [http://localhost:8095/swagger/views/swagger-ui#/](http://localhost:8095/swagger/views/swagger-ui#/)

If swagger-ui don't open in the browser, delete de build folder in the MS root and build the project again.
