# ms-firebase

## How to run:

1. Add the correct file - "firebase-credentials.json" and change in the application.yml for your local path
2. Find all instances of "CHANGE_ME" and replace with correct variables

## How to get "firebase-credentials.json":

1. Open the console of the project

![Console](\src\main\resources\readme-images\image_1.png)

2. Click in the project name

![Console](\src\main\resources\readme-images\image_2.png)

3. Click on the Definitions

![Console](\src\main\resources\readme-images\image_3.png)

4. Go to account settings

![Console](\src\main\resources\readme-images\image_4.png)

5. Select the image

![Console](\src\main\resources\readme-images\image_5.png)

6. Click on "Generate new private key"
