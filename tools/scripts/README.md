# Mobile

<details>
<summary>Rename app</summary>

In order to rename the app, we must replace some values to fit our project.
<br>
Search for every occurence of the following terms and change them for the name of the app (without spaces):

- change_me_mobile (respect the _ )
- change me mobile (if the app name has space, this is the place to apply them)
- changememobile (no spaces, all together)

After that, we need to do a last change in the android project.
<br>
Go to the folder in `apps/client/mobile/android/app/src/main/kotlin/pt/atlanse` and change the folder `change_me_mobile` to the app name following the same pattern (keep the _).

</details>
